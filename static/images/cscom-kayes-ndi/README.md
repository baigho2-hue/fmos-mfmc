# Images du CSCom de Kayes N'di

Ce dossier contient les photos du CSCom de Kayes N'di pour la section "Médecine de Famille et Médecine Communautaire" de la page d'accueil.

## Instructions pour ajouter des images

1. Placez vos images dans ce dossier (`static/images/cscom-kayes-ndi/`)
2. Nommez-les selon le format : `cscom-1.jpg`, `cscom-2.jpg`, `cscom-3.jpg`, etc.
3. Formats supportés : `.jpg`, `.jpeg`, `.png`
4. Le carousel affichera automatiquement toutes les images disponibles

## Nombre d'images

Le carousel est configuré pour afficher jusqu'à 5 images (cscom-1.jpg à cscom-5.jpg). Si vous avez plus d'images, vous pouvez :
- Modifier le fichier `core/templates/accueil.html` pour ajouter plus de slides
- Ou remplacer les images existantes

## Notes

- Les images doivent être optimisées pour le web (taille raisonnable)
- Dimensions recommandées : largeur minimale de 800px
- Le carousel utilise une rotation automatique toutes les 5 secondes
- Si une image ne se charge pas, l'image par défaut sera affichée
