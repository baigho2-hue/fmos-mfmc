# Dossier de Configuration

Ce dossier contient les fichiers de configuration du projet.

## Fichiers de configuration

- `requirements.txt` - Dépendances Python du projet
- `Procfile` - Configuration pour le déploiement (Railway, Render, Heroku)
- `runtime.txt` - Version de Python utilisée
- `cours_extracted.json` - Données de configuration des cours
- `cours_extracted_improved.json` - Données améliorées des cours

## Note importante

Les fichiers `requirements.txt`, `Procfile` et `runtime.txt` doivent également être présents à la racine du projet pour que les plateformes de déploiement (Railway, Render, etc.) puissent les détecter automatiquement.

Ces fichiers sont donc dupliqués à la racine pour le déploiement et conservés ici pour l'organisation du projet.
